Multimodal feature fusion for gait recognition
==============================================

F.M. Castro and M.J. Marin-Jimenez

Contents
~~~~~~~~

This package contains:
 - Matlab code for computing the audio-visual descriptors proposed in [1].

Prerequisites
~~~~~~~~~~~~~
 * VLFeat library >=0.9.19 for Matlab: http://www.vlfeat.org/
 * Dense tracks software from: http://www.irisa.fr/texmex/people/jain/w-Flow/
 * MIR toolbox >=1.5 from https://www.jyu.fi/hum/laitokset/musiikki/en/research/coe/materials/mirtoolbox

Quick start
~~~~~~~~~~~

1) Start Matlab
2) cd <pfmrootdir>
3) addpath(genpath(pwd))
4) demofusion
